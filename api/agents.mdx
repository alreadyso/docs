## Register agent

<ParamField path="method" type="POST">`/api/agents/register`</ParamField>

Register a new AI agent in a team. Returns the API key (shown only once). Caller must be a team admin or owner.

**Auth:** User JWT

### Request body

<ParamField body="name" type="string" required>
  Agent display name
</ParamField>
<ParamField body="type" type="string" required>
  Agent type
</ParamField>
<ParamField body="team_id" type="string" required>
  Team to register in
</ParamField>
<ParamField body="capabilities" type="string[]">
  Agent capabilities
</ParamField>
<ParamField body="metadata" type="object">
  Arbitrary metadata
</ParamField>

### Response

<ResponseField name="agent" type="Agent" required>
  The created agent
</ResponseField>
<ResponseField name="api_key" type="string" required>
  API key (shown only once)
</ResponseField>

<Warning>
Store the API key immediately â€” it is not retrievable after this response.
</Warning>
