## Create client secret

<ParamField path="method" type="POST">`/api/realtime/client-secret`</ParamField>

Create an OpenAI Realtime API client secret for transcription or realtime sessions.

**Auth:** Agent key or User JWT + X-Team-Id

### Request body

<ParamField body="session_id" type="string">
  Session ID to associate (validated against team)
</ParamField>
<ParamField body="session_type" type="string">
  `transcription` or `realtime` (default: `transcription`)
</ParamField>
<ParamField body="model" type="string">
  OpenAI model to use
</ParamField>
<ParamField body="language" type="string">
  Language code
</ParamField>
<ParamField body="prompt" type="string">
  System prompt
</ParamField>
<ParamField body="expires_in_seconds" type="number">
  TTL in seconds (10-7200, default 60)
</ParamField>

### Response

<ResponseField name="client_secret" type="object" required>
  Object with `value` (string) and `expires_at` (timestamp)
</ResponseField>
<ResponseField name="session" type="object" required>
  OpenAI session config
</ResponseField>
<ResponseField name="session_type" type="string" required>
  The session type
</ResponseField>
<ResponseField name="team_id" type="string" required>
  Team ID
</ResponseField>
<ResponseField name="session_id" type="string">
  Associated session ID
</ResponseField>
